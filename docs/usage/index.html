使用

假设有本地git工作区/tmp/workdir

#进入本地仓库

cd /tmp/workdir

#初始化远程透明加密配置

第一次使用gitc,需要对仓库进行初始化操作.

gitc cryptinit

或者:

gitc cryptinit -p your_passwd

远程透明加密支持两种加密方式:

  1. 自动生成key文件
  2. 密码

在加密和解密时不需要人工干预. 需要保管好key文件

#解密仓库

如果新克隆的仓库已经被加密, 需要一次解密操作.

gitc unlock -p your_passwd

或者:

gitc unlock -k key_path

仅仅需要一次解密操作!

#将文件标记为透明加密

gitc encrypt -r plain.md

#将文件标记为本地加密

gitc encrypt plain.md -l -p passwd_for_this_file

#本地解密

gitc decrypt plain.md -l -p passwd_for_this_file

#远程解密

gitc decrypt -r plain.md
